# ShopDrop

A Rust-based application for managing shopping lists and inventory.

## Features

- Manage shopping items
- Track inventory
- Simple and efficient data storage

## Installation

Ensure you have Rust installed. Then clone the repository and build the project:

```bash
git clone https://github.com/russ021/shopdrop_app.git
cd shopdrop_app
cargo build --release
```

## Usage

Run the application:

```bash
cargo run
```
